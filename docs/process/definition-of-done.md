# Definition of Done (DoD)

## 1) Scope
- Matches issue objective and acceptance criteria

## 2) Design QA
- Uses brand tokens from docs/brand
- Includes before/after screenshots

## 3) Functional QA
- Key flow works end-to-end
- Form validation + empty/error states

## 4) Engineering QA
- lint/test/build pass
- no obvious regression

## 5) Business QA
- CTA clarity and tracking event present

## PR decision states
- ACCEPT
- ACCEPT_WITH_FOLLOWUPS
- REJECT
